# คู่มือการติดตั้งและตั้งค่าโปรเจค DUYดูDEE (Installation & Hosting Setup)

ยินดีต้อนรับ! นี่คือคู่มือขั้นตอนการติดตั้งโปรเจคนี้บนเครื่องคอมพิวเตอร์ใหม่ เพื่อให้มั่นใจว่าคุณสามารถรันและ Deploy โปรเจคได้อย่างถูกต้องครับ

---

## 1. ติดตั้งเครื่องมือที่จำเป็น (Prerequisites)
ก่อนเริ่มต้น ให้ติดตั้งเครื่องมือเหล่านี้ในเครื่องของคุณก่อน:

1. **Node.js:** ดาวน์โหลดและติดตั้ง [Node.js เวอร์ชัน LTS](https://nodejs.org/) (ช่วยให้เราสามารถรัน NPM สำหรับจัดการ Library ต่างๆ ได้)
2. **Firebase CLI:** นี่คือเครื่องมือสำหรับ Hosting และจัดการ Firebase ของเรา ให้เปิด CMD หรือ Terminal แล้วรันคำสั่ง:
   ```bash
   npm install -g firebase-tools
   ```
3. **Login เข้าสู่ Firebase:** รันคำสั่งนี้เพื่อเชื่อมต่อเครื่องของคุณกับบัญชี Firebase ของคุณ:
   ```bash
   firebase login
   ```

---

## 2. การตั้งค่าโปรเจค (Setup)
เมื่อได้เครื่องมือครบแล้ว ให้ทำตามขั้นตอนนี้ในโฟลเดอร์โปรเจค:

1. **ติดตั้ง Library:** เปิด CMD ในโฟลเดอร์โปรเจค (`D:\DUYDODEE-HD`) แล้วรันคำสั่ง:
   ```bash
   npm install
   ```
   *(ขั้นตอนนี้จะโหลด Library ทั้งหมดที่จำเป็นในการรันเว็บมาไว้ใน `node_modules`)*

2. **เชื่อมต่อโปรเจคกับ Firebase:**
   ```bash
   firebase use duydodeesport
   ```

---

## 3. การรันและการ Deploy (Deployment & Testing)

โปรเจคนี้ถูกออกแบบมาให้จัดการง่ายผ่านสคริปต์อัตโนมัติ:

*   **Deploy เว็บไซต์ (ทำบ่อย):** ใช้สคริปต์ `Ship.bat` ที่เราเตรียมไว้ให้
    *   เพียงดับเบิลคลิกไฟล์ `Ship.bat` หรือพิมพ์ใน CMD ว่า `.\Ship.bat`
    *   สคริปต์จะตรวจสอบความถูกต้องของโค้ด, รัน Unit Test, และ Deploy ขึ้น Firebase ให้ทันที
    
*   **การทดสอบโค้ด:** หากต้องการตรวจสอบ Logic หลักว่าทำงานถูกต้องไหม ให้รัน:
    ```bash
    npm test
    ```

---

## 4. ข้อควรจำ
*   **อย่าลบ `package-lock.json` หรือ `node_modules`** โดยไม่จำเป็น หากต้องการล้างโปรเจคให้สะอาดที่สุด ให้ลบโฟลเดอร์เหล่านั้นแล้วรัน `npm install` ใหม่เท่านั้น
*   หากเปลี่ยนเครื่องใหม่ และต้องการแก้ไขโค้ด อย่าลืมอ่านไฟล์ `DEVELOPER_GUIDE.md` เพื่อทบทวนมาตรฐานการเขียนโค้ดครับ
