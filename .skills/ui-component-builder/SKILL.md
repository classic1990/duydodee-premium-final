# UI Component Builder (DUYDODEE-HD Edition)

Skill นี้ช่วยสร้างส่วนประกอบหน้าเว็บใหม่ๆ ให้สอดคล้องกับ Design System ของ DUYDODEE-HD

## Design Principles
- **Aesthetic:** เน้นโทนสีเข้ม (Brand Black), ตัดด้วยสีทอง (Brand Gold/Primary), และใช้ Backdrop Blur
- **Typography:** ใช้ฟอนต์ 'Thai-font' สำหรับภาษาไทย และเน้นความหนา (Black/Bold) สำหรับหัวข้อ
- **Interactivity:** มี Hover effects, Smooth transitions, และ Glassmorphism effects

## Component Patterns
- **Movie Cards:** ดูแม่แบบที่ [assets/templates/movie-card.html](assets/templates/movie-card.html)
- **Sections:** ใช้ระยะห่าง (Spacing) และการจัดวาง (Grid/Flex) ตามมาตรฐานใน `public/css/styles.css`
- **Icons:** ใช้ Lucide Icons เท่านั้น

## Resources
- **Design System:** [references/design_system.md](references/design_system.md) (ประกอบด้วย Code colors และ Classes ที่ใช้บ่อย)
